Checkboxes can be grouped to act like radio inputs.



Or they can act like normal checkboxes.

Three types of callbacks: Checked, unchecked, and changed. Be careful as on checked and on change can be called at the same time. Only use onChange when you need to know if the user changed an option. The callbacks get passed an object containing a variable 'changedItem','data.unCheckedItem', or 'data.checkedItem' which is the id of the div container that was checked.
